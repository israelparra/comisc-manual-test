1. Descripción del Sistema

OrderFlow es un microservicio Spring Boot que gestiona usuarios, productos y pedidos de una tienda online. Expone una API REST con cinco controladores independientes.

  • Tecnología: Spring Boot con Spring Data JPA y OpenFeign para comunicación entre servicios.
  • Dominio: Usuarios, Productos, Pedidos, Reportes y Fulfillment de una tienda online.
  • Arquitectura: Controladores REST → Repositorios JPA / Feign Clients → Base de datos / Servicios externos.
  • Tu tarea: Medir el CFP de cada uno de los 17 endpoints usando el video tutorial y la CheatSheet COSMIC.
No necesitas ejecutar ni compilar el proyecto. Analiza el código fuente directamente en esta página.
Al final de cada controlador encontrarás una tabla donde debes registrar tu medición (E, X, L, W y CFP). Al terminar todos los controladores, introduce tu nombre y envía los resultados.
